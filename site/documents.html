A collection of some basic documentation for eSite.

An Introduction to eSite
A brief introduction to eSite including a minimal Hello World example.
Code Transformations While Parsing
An explanation of what eSite does to a document to make it possible to parse the output with PHP.
Functions and Variables
Functions and variables that are defined for use by website developers within the eSite parser.

Note that the eSite parser is still so young a product that it had to be tweaked to be able handle its own documentation without choking on the parser or requiring a readable reference of the legal XHTML escaped entities. In any event, all contents of this site were produced using the eSite parser.
