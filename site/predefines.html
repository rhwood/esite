[ESITE TITLE Functions and Variables] [ESITE BEGIN]

eSite defines some functions and variables that may be used by website developers. No function or variable named esite_* is guarantied to not be already used by the eSite parser. Functions and variables named esite_internal_* are reserved for eSite internal use and, in the case of functions, are not guarantied to maintain the same behavior between eSite versions, and, in the case of variables, not to be modified by eSite functions in unexpected manners.

Classes

eSite_Parser
The class definition for the eSite website parser.

Functions

esite_debug($flag = "print_debugging_statement")
Returns TRUE if one of the eSite initialization scripts set the desired flag (which defaults to print_debugging_statement). In all other cases, it returns FALSE
esite_exec($program, $parameters, $path = '')
Execute the program $program with the given parameters $parameters. If $path is set, $program must exist in $path, otherwise, it must exist in the system $PATH.
esite_parse_content($esite_content)
Parse the contents of a variable as if the variable contained a set of valid PHP code.
esite_setting($section, $setting)
Return the value of the setting $setting in section $section. These values are all set in esite.ini initialization file.

Objects

&esite_parser
The eSite parser object. This object does the work of parsing the templates and sources of the website. It is an instance of eSite_Parser.

Variables

$esite_content
The content of a document, provided [--ESITE BEGIN] and [--ESITE END] bracketed statements exist in the document.
$esite_ini
Array of configuration values set when eSite parser is initialized.
$esite_root
The root directory of the website. This needs to be set to some value such that it can be used to navigate through the website. The default value is . This value is best set to either a relative directory or to a valid absolute URL.
$esite_title
The title of a document. This value can be set either with the [--ESITE TITLE some document title] bracketed statement or in code.
$esite_write_file_to_name
The name of the file to be written to the website. By default this is the same as the name of the file being parsed. This value can be set either with [--ESITE CHANGE NAME TO filename.extension] bracketed statement or in code. This value may be set at any point within the file being parsed.
