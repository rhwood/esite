About eSite

eSite is a tool for building complex, templated, static websites without having to rely on expensive propriety tools. eSite uses a parser, esite, to parse pages based on a PHP-like syntax.

News

The eSite project has been restarted with new goals and aims. This is a birthday present to myself.

Goals

  • Provide an easy-to-use parser for files that uses PHP language constructs in an XML and PHP compliant manner.
